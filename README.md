# The Political Research Based on Ancient Music DJ Comments

This repository stores the sanitized course-paper materials and aggregate data tables for a BPCR-based exploratory study of Bilibili ancient-style music comment sections.

## Contents

- `docs/course_paper.md`: course paper draft in Chinese.
- `data/aggregate_summary.json`: sanitized aggregate counts and research metadata.
- `data/song_level_summary_sanitized.csv`: per-song comment counts and political/historical related shares.
- `data/stance_distribution_sanitized.csv`: stance distribution recoded into broad categories.
- `data/theme_distribution_sanitized.csv`: first-level theme distribution only.
- `data/DATA_SOURCE.md`: data source, processing method, and ethics notes.

## Privacy and sensitivity handling

This public repository intentionally excludes raw comments, user IDs, usernames, profile links, cookies, API keys, and second-level labels that may contain specific sensitive names or events. The uploaded files are aggregate and sanitized artifacts for course-paper reproducibility only.
